# Identity Based Healthcare System with Security Features

**Installation Steps**

1. **Start XAMPP** → Turn ON **Apache** and **MySQL**
2. **Create Database** → Open phpMyAdmin and create database:  
   `onenationoneidentity_cs` then import the provided SQL file
3. **Open System** → In your browser, go to:  
   `http://localhost/onenationoneidentity-fixed`

**System Access**

- **Main System:**  
  `http://localhost/onenationoneidentity-fixed`  
  *(Manage doctors, patients, hospitals, etc.)*

- **Admin Panel (Security Features):**  
  `http://localhost/onenationoneidentity-fixed/admin`

**Security Features Included**

1. **User Authentication** – Secure login system
2. **Real-Time Admin Alert System** – Instant security notifications
3. **Two-Step Verification** – Email verification for login
4. **CAPTCHA Protection** – Prevents bot attacks
5. **Session Timeout** – Auto logout after inactivity
6. **Role-Based Access Control** – Different permissions for different users
7. **Data Access Control** – Tracks who accesses what data
8. **Secured Password Policy** – Strong password requirements
9. **Account Lockout** – Locks account after failed attempts
